# Catch The Falling Objects

## Overview

Catch The Falling Objects is a simple arcade-style game developed using Python and Tkinter. The player controls a basket using the keyboard and must catch falling objects to earn points. The game tests the player's reflexes and hand-eye coordination while providing an engaging graphical interface.

## Features

- Built entirely using Python and Tkinter
- Keyboard-controlled basket movement
- Real-time score tracking
- Random object spawning
- Game Over detection
- Simple and beginner-friendly code structure
- Interactive graphical user interface

## Technologies Used

- Python 3
- Tkinter
- Random Module

## How to Play

1. Run the Python file.
2. Use the **Left Arrow Key** to move the basket left.
3. Use the **Right Arrow Key** to move the basket right.
4. Catch the falling objects with the basket.
5. Each successful catch increases your score.
6. Missing an object ends the game.

## Controls

| Key | Action |
|------|---------|
| ← Left Arrow | Move Basket Left |
| → Right Arrow | Move Basket Right |
